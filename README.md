# KEEPCASE 行李旅程事件簿

依教育部大專校院創新創業實戰模擬學習平臺 111 年成功案例「SuperTony—『無鎖不在』外出行李箱防盜感測器」延伸的概念原型。

## 核心流程

1. 綁定航班、行李牌、外觀照片與近距離感測器。
2. 偵測移動與藍牙斷線後，先由旅客判斷是否為正常情境。
3. 找不到行李時啟用遺失模式，保留事件時間線。
4. 立即向航空公司或機場行李服務櫃檯申報並記錄參考編號。
5. 整理行李牌、照片、事件、證明與收據，產生申報資料包。

## 產品與市場

- 核心使用者與痛點：自由行旅客、家庭與旅行團在行李異常時，常缺少完整紀錄、申報順序與證明資料。
- 早期客群與場景：先從高頻出國旅客、旅行社團體與行李品牌售後服務切入。
- 市場差異：不是單純藍牙警報，而是把行前準備、情境確認、航空申報和資料包串成一條服務流程。
- 導入與採購：以機場接送、旅行社或行李品牌場域試辦，觀察警示誤報、客服負擔與旅客回饋，再決定採購或聯名銷售。
- 旅客購買低功耗感測器。
- 年度會員提供多行李、事件雲端備份、家庭共享與資料包。
- 與旅行社、保險與行李品牌合作，形成 B2B2C 售後服務。
- 驗證指標包含警示誤報率、旅程啟用率、事件紀錄完整率、申報資料完整率、尋回或申報成效、試辦回饋、會員續訂與硬體退貨率。

## 邊界

本產品不是 GPS、不是航空公司官方追蹤、不是報案或理賠系統，也不保證防盜、尋回或獲賠。感測事件不可作為單獨失竊證明；所有航班、行李與事件均為模擬。
